OVROverlay Draw order

Tue, Jul 3, 2018

We want render clear text. we need use Oculus overlay

In order what OVROverlay does. that talk about the normal drawing in Unity.

Drawing order in Unity

  1. opacity geometry
  2. skybox
  3. transparent

now, we have eye-buffer texture.

It’s now Oculus’s part to render this texture on screen.

OVROverlay

=======

Draw order

Gala360:

Orthographic camera – render – UI Canvas => UI Texture

UI Texture -> OVR Overlay -> draw as [Overlay Cylinder] 0 Cursor -> OVR Overlay -> draw as [Overlay Quad] 1 Skybox -> OVR Overlay -> draw as [Underlay Cubemap] 2

====== With Laser v1 ======

Laser -> draw as [Unity eyetexture] UI Texture => draw as [Underlay Cylinder] Skybox -> draw as [Underlay Cubemap]

====== With Laser v2 ======

UI Texture => draw as [Overlay Cylinder] Skybox -> draw as [Underlay Cubemap]

Laser -> draw as [Overlay ??]
